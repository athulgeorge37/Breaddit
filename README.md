# Breaddit A FullStack Reddit Clone:

Stack:

-   React
-   Nodejs
-   Express
-   MySql
-   Sequelize
-   Graphql

Other Tools:

-   Rest APIs
-   Subscriptions
-   JSON Web Tokens
-   Authentication

## Installation

cd into server and client folder and run:

```
npm install
```

followed by:

```
npm start
```
